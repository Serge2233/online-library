# Online Library

## Purpose

Showcase of a book search engine built with MERN stack, utilizing Google Book search API as well as GraphQL implementation. Allows users to sign up/log in to search and save their favorite books.

## Built with

* React
* MongoDB
* Express
* Node.JS
* GraphQL
* CSS
* Bcrypt
* Mongoose
* JWT
